# daily.dev Documentation Site

This is the official documentation website for daily.dev, built with Docusaurus 3.

## About daily.dev
daily.dev is a professional network for developers to stay updated with the latest tech news, discover new tools, and connect with peers. It aggregates content from hundreds of sources and uses AI to personalize feeds for each developer.

## Project Structure

### Framework & Technology
- **Docusaurus 3**: Static site generator with React-based theming
- **React 18**: Component framework for interactive elements
- **MDX**: Markdown with JSX support for rich documentation
- **Algolia**: Integrated search functionality
- **Google Analytics**: User behavior tracking

### Key Directories
- `docs/`: All documentation markdown files organized by category
- `src/components/`: React components for homepage and custom elements
- `src/pages/`: Custom pages outside the docs structure
- `static/`: Static assets including images, icons, and manifest files
- `docusaurus.config.js`: Main site configuration
- `sidebars.js`: Auto-generated sidebar structure

The documentation covers:

### Development Commands
- `npm start`: Development server (localhost:3000)
- `npm run build`: Production build
- `npm run serve`: Serve built site locally
- `npm run clear`: Clear Docusaurus cache

### Key Features
- Dark mode by default with light mode toggle
- Responsive design optimized for all devices
- Image optimization with @docusaurus/plugin-ideal-image
- Direct GitHub edit links for all documentation
- PWA capabilities with offline support
- SEO optimized with structured data

### Major Features Documentation

**Career Mode** (`/docs/career-mode/`)
A developer-first hiring experience that helps developers land their next job on their own terms. Key aspects:
- Private by default - no recruiter spam
- Double opt-in matching system between developers and employers
- Curated high-quality job opportunities based on skills, interests, and preferences
- Complete transparency with job opportunity briefs
- Free forever for developers
- Comprehensive coverage of job preferences, profile setup, matching algorithm, privacy model, and FAQ

**Organizations** (`/docs/organizations/`)
Team subscription management for companies to provide daily.dev Plus to their engineering teams.

**Monetization** (`/docs/monetization/`)
In-app currency (Cores), Awards, and Boost features for content creators and community engagement.

**Squads** (`/docs/squads/`)
Private and public developer communities for sharing content and discussions.

**Plus Features** (`/docs/plus/`)
Premium features including Presidential Briefing, Smart Prompts, Advanced Custom Feeds, Clickbait Shield, and more.
